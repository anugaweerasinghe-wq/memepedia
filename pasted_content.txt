## 🎨 VISUAL DESIGN REQUIREMENTS

### Color Palette (use CSS variables):
- --bg: #050508 (near-black with blue undertone)
- --surface: rgba(255,255,255,0.04)
- --surface-hover: rgba(255,255,255,0.08)
- --border: rgba(255,255,255,0.10)
- --accent-1: #FF3CAC (hot pink)
- --accent-2: #784BA0 (deep purple)
- --accent-3: #2B86C5 (electric blue)
- --text-primary: #F5F5F5
- --text-muted: rgba(255,255,255,0.45)
- Gradient mesh background: animated radial gradients shifting 
  between the three accent colors at 20s cycle

### Typography:
- Headlines: "Bebas Neue" (Google Fonts)
- Body / UI: "DM Sans" (Google Fonts)  
- Accent / quotes: "Playfair Display Italic" (Google Fonts)
- Year numbers: 20vw+ size, ultra-light weight, letter-spaced, 
  used as design elements

### Glassmorphism Rules:
- All cards: background rgba(255,255,255,0.04), 
  backdrop-filter blur(16px) saturate(180%), 
  border 1px solid rgba(255,255,255,0.10),
  border-radius 16px, box-shadow with accent color glow

### Animations (ALL must be implemented):
1. **Parallax scrolling**: Year numbers scroll at 0.3x speed 
   relative to content (CSS transform translateY with JS scroll listener)
2. **Intersection Observer reveals**: Every card, stat, and section 
   fades + slides up when entering viewport (staggered 100ms delays)
3. **Animated gradient mesh background**: 3 radial gradient orbs 
   that slowly drift using @keyframes (no libraries needed)
4. **Card hover effects**: 
   - translateY(-8px) lift
   - Border glow pulses in accent color
   - Emoji scales up 1.2x
   - Shimmer sweep animation across card surface
5. **Counter animations**: Meme count numbers animate from 0 to 
   their value using requestAnimationFrame when scrolled into view
6. **Hero text glitch**: CSS glitch animation on "MEMEPEDIA" 
   using ::before ::after pseudo-elements with clip-path
7. **Scroll progress bar**: Thin gradient line at top of page 
   showing how far user has scrolled
8. **Cursor glow**: Custom cursor with a soft radial glow that 
   follows mouse movement (div with pointer-events:none)
